# Browser Screenshots

To create browser screenshots of the web UI:

1. `openosint web`
2. Open http://localhost:8080
3. Use CMD+Shift+4 (Mac) to screenshot
4. Save to `media/screenshots/`

Suggested shots:
- `empty-state-dark.png` — empty state, dark theme
- `empty-state-light.png` — empty state, light theme (click sun icon in top bar to toggle)
- `active-investigation.png` — mid-investigation with tool cards visible
- `tool-card-expanded.png` — tool output card expanded showing results
- `settings-modal.png` — settings modal open with API key fields
